Keyboard Shortcuts¶

CryptPad provides a variety of keyboard shortcuts to help you navigate and interact with the platform more efficiently.

General Shortcuts¶

Shortcut

Action

Ctrl + E

Create a new document

Tab

Navigate through the focusable elements

Shift + Tab

Navigate backwards through the focusable elements

Enter

Click on the focused element

Drive Shortcuts¶

Shortcut

Action

↑, ↓, ←, → keys

Navigate through content in the drive

Enter

Open drive file or folder

Toolbar Shortcuts¶

Shortcut

Action

Enter

Click on the focused element to open a menu

↑ and ↓ keys

Navigate through items inside a menu

Esc

Close menu

Del

Remove notification from the notifications menu

Search Functionality

Locate specific elements within a menu by typing the first letter

Apps Shortcuts¶

Shortcut

Action

Esc

Remove focus from CKEditor4 to avoid keyboard trap

For shortcuts specific to third-party applications, please refer to their respective documentation:

  • Drawio shortcuts

  • CKEditor shortcuts

  • OnlyOffice shortcuts
