Código / Markdown¶

El Código/Markdown en CryptPad es una integración de CodeMirror.

Barra de herramientas¶

Herramientas: Muestra/oculta la barra de herramientas del editor de texto.

Insertar: Añade una imagen al documento. La imagen se puede elegir en el disco o cargar. Logged in users

Tema: Establecer colores en el editor, detallado abajo.

Vista previa: Mostrar/ocultar el panel de vista previa de Markdown.

Tema¶

Color por autor: Resalta el texto escrito por cada usuario con el color de su cursor (elegido en configuración de usuario). Cuando está activo:

  • Ocultar colores de autor para desactivar la visualización de colores en esta ventana. Los colores se pueden volver a activar y seguirán estando activos para otros usuarios.

  • Color por autor > Borrar y desactivar para desactivar los colores para todos los usuarios y eliminar los datos.

Tema: Combinación de colores utilizada en el panel del editor de código.

Idioma: Se usa para resaltar la sintaxis.

Importar/Exportar¶

Archivo > Importar.
Formatos admitidos: Cualquier archivo del texto sin formato, la extensión del archivo se utiliza para determinar el idioma.
Archivo > Exportar.
Formatos compatibles: .md, cualquier otra extensión puede ser escrita para el nombre del archivo exportado.

Markdown¶

La aplicación Code es particularmente adecuada para escribir documentos en Markdown: una sintaxis liviana que ofrece formatos básicos sin dejar de ser legible. Markdown se convierte fácilmente a otros formatos como HTML.

Para ver ejemplos de sintaxis de Markdown en CryptPad, consulta:

Markdown Guide: Basic Syntax

Además de las sintaxis básicas de Markdown, también están disponibles las siguientes funciones:

  • [TOC] Inserta una tabla de contenido.

  • Hacer la lista

    • - [ ] tarea para una tarea pendiente: tarea.

    • - [x] tarea para una tarea completada: tarea.

Así como la siguiente extensión:

  • Diagramas con Mermaid.

  • Mapas mentales con`Markmap <https://markmap.js.org/>`__.

  • Ecuaciones matemáticas con Mathjax.

Para usar una extensión, asígnale el nombre de sintaxis para un bloque de código Markdown, por ejemplo:

``` mermaid
   <!-- your mermaid code -->
```

Para ver ejemplos detallados de las extensiones en uso, consulta:

Markdown Guide: Extensions

Imágenes¶

La sintaxis Markdown para incluir imágenes remotas, como `` ![description](https://site.com/image.jpg) `` está bloqueada por CryptPad por razones de seguridad.

Usuarios registrados

Para insertar imágenes desde el CryptDrive o subir nuevas, usa el menú Insertar en la barra de herramientas.

Consulta Contenido remoto para obtener más detalles.

Lightbox¶

Para explorar todas las imágenes y diagramas de un documento utilizando todo el ancho de la ventana:

  1. Doble click en una imagen o diagrama en la vista previa de Markdown.

  2. Navega con las teclas de flecha.

  3. Cierra la Lightbox con o Esc.
