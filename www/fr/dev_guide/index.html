Developer Guide¶

Welcome to the CryptPad Developer Guide.

This guide explains how to set up a local development environment, the inner workings of CryptPad, and how to contribute to the project.

  • Set up a development instance
  • Contribute Code
  • Key elements in CryptPad
  • Database
  • Code

Note

Please note that due to its length and technical nature this guide is not part of the translations for the CryptPad documentation. It is presented in English only. This is to avoid discouraging translators willing to work on the more widely useful Guide Utilisateur·ice by adding a high volume of technical text.
