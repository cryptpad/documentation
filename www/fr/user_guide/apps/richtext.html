Documents texte¶

L'éditeur de texte CryptPad est une intégration de CKEditor.

Barre d'outils¶

Outils : Affiche ou cache la barre d'outils de l'éditeur de texte.

Insérer : Intègre une image dans le document. L'image peut être choisie dans le drive ou importée. Utilisateurs enregistrés

Paramètres d'affichage¶

Les boutons en haut de l'éditeur contrôlent l'affichage des informations de l'éditeur de texte.

: Afficher/masquer le sommaire du document.

: Afficher/masquer les commentaires (une icône rouge indique qu'il y a des commentaires).

: Utiliser toute la largeur de la fenêtre pour l'éditeur de texte.

: Utiliser le "mode page" à largeur réduite pour l'éditeur de texte.

Pour définir des valeurs par défaut pour ces paramètres, par exemple pour contrôler quelles sections sont visibles lorsque les destinataires visitent un lien partagé vers le document, utilisez le menu Paramètres du document :

Fichier > Paramètres du document.

Écriture¶

Titres¶

Pour que les titres apparaissent dans le sommaire, ils doivent être formatés comme tels. Pour mettre en forme un titre, sélectionnez le texte ou placez le curseur sur la ligne du titre, puis soit :

  • Utiliser le menu de formatage de la barre d'outils (affiche "normal" par défaut) pour sélectionner le niveau d'en-tête souhaité.

  • Utiliser le raccourci clavier : Ctrl + Alt + le numéro du niveau de titre souhaité.

Paragraphes¶

Par défaut, la touche Entrée crée un nouveau paragraphe, ajoutant une ligne de séparation avec le paragraphe précédent. Pour ajouter un saut de ligne sans ajouter un nouveau paragraphe, utilisez Maj + Entrée.

Note

Veuillez jetez un œil à la`documentation CKEditor <https://ckeditor.com/docs/ckeditor4/latest/features/shortcuts.html>`__ pour une liste complète des raccourcis clavier disponibles.

Importer/Exporter¶

Fichier > Importer
Formats supportés : .html, .md.
Fichier > Exporter
Formats supportés : .html, .doc, .md.

Pour imprimer ou exporter au format PDF :

  1. Fichier > Imprimer.

  2. Sélectionner une imprimante ou un ficher PDF comme destination.

Commentaires¶

Pour commenter une partie du texte :

  1. Sélectionner le texte à commenter.

  2. Bouton , à droite du texte ou dans la barre d'outils de l'éditeur texte.

  3. Écrire le commentaire.

  4. Valider.

Cliquer sur un commentaire pour le sélectionner, puis :

  • Répondre : Ajoute un commentaire en réponse.

  • Fermer : Supprime le commentaire après confirmation.

Pour envoyer une notification à un·e autre utilisateur·ice, vous pouvez les mentionner avec @ et leur nom affiché.

Vous recevrez des notifications dans les cas suivants :

  • Quelqu'un répond à l'un de vos commentaires.

  • Quelqu'un vous mentionne.
