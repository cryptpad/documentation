Часто задаваемые вопросы¶

Общие проблемы¶

Если CryptPad не работает должным образом, можно попробовать несколько распространенных способов. Большинство из них относятся к веб-браузерам и расширениям, которые либо мешают функциям безопасности CryptPad, либо отключают основные функции браузера.

  1. Проверьте список расширений ниже, отключите те, которые Вы установили, если они есть

  2. Попробуйте использовать браузер в режиме приватного окна и/или с отключенными расширениями

  3. Try using a different browser. We recommend the latest versions of Firefox or Chromium

  4. If the issues persist, please contact your instance Техподдержка

Список расширений, которые, как известно, вызывают проблемы с CryptPad¶

  • Librejs

  • Adblock

  • Night Eye

  • uMatrix

  • JShelter

  • AdGuard

  • SuperAgent

  • Tampermonkey

  • Arkenfox

  • Loom

  • MozBar

  • Stands AdBlocker

  • Emoji keyboard emojis for chrome

  • PAYBACK Internet Assistent (PIA)

  • User-Agent Switcher and Manager

Примечание

If you identify issues with another extension please report it to the development team

Я забыл свой пароль¶

Администраторы экземпляра CryptPad не могут восстановить или сбросить пароли пользователей. Это также гарантирует, что они не смогут получить доступ к каким-либо пользовательским данным (например, к содержанию документов).

Если Вы забыли пароль для своей учетной записи, но всё ещё авторизованы в системе, вы можете перенести свои данные в новую учетную запись выполнив следующие шаги:

  1. Находясь в учетной записи от которой утерян пароль, перейдите к: Меню пользователя (аватарка в правом верхнем углу) > Настройки > CryptDrive

  2. Нажмите кнопку Резервное копирование, чтобы скачать файл, содержащий ключи ко всем документам. (примечание: это отличается от Скачать мой Cryptdrive, которое сохраняет содержимое всех отдельных документов)

  3. В другом браузере зарегистрируйте новую учетную запись с известным паролем

  4. While logged in to the new account: User Menu (avatar at the top right) > Settings > CryptDrive and click the Restore button

How to import/export to/from another platform?¶

Making CryptPad compatible with a wide range of document formats is an ongoing challenge for the development team. While work in this area is in progress, the methods below should help in importing and exporting to/from other widely used software.

Примечание

Простое копирование-вставка содержимого из существующего программного обеспечения в CryptPad - это простой первый шаг при импорте/экспорте. Пожалуйста, обратите внимание, что приведенные ниже методы импорта/экспорта могут значительно улучшить результаты.

It is possible to upload files to CryptPad for secure storage and sharing, however this does not make every file an editable CryptPad document right away. For this, you either have to:

  • Right click on the uploaded file > Open in editor, or...

  • Create a blank document and use the File > Import functionality in the appropriate CryptPad editor as explained below

Предупреждение

  • The Safari web browser, used on macOS, iPadOS and iOS, isn't supported at this time

  • Support for CSV is currently broken, .csv import/export is disabled for the time being

  • Support for OpenDocument formats, .odt, .odp & .ods import/export is currently experimental and prone to errors

If you encounter any issues, please check the Общие проблемы first. If you aren't able to make it work following these instructions, please reach out to your instance administrators through the Техподдержка help desk. Providing an example document with the Поделиться ссылкой functionality.

Импорт/экспорт не отображается в меню «Файл»¶

Для работы с электронными таблицами и расширенной обработки документов, Cryptpad интегрирует OnlyOffice. В результате на одном экране появляются две панели инструментов, каждая со своим собственным меню «Файл». Пожалуйста, обязательно проверьте оба меню при поиске авриантов импорта/экспорта.

Document application import/export¶

The Документ supported format are Microsoft Word .docx and OpenDocument .odt. These can both be exported from popular software such as Google Docs and LibreOffice Writer:

  1. Export your document as .docx or .odt in your existing software

  2. Save the file to your computer

  3. Open a new Документ file (or an existing one to replace the contents)

  4. File > Import and select your file

Для экспорта используйте Файл > Экспорт и выберите формат:

  • To use in Microsoft Word or Google Docs, select .docx

  • To use in LibreOffice Writer, select .odt

  • For cases where no further editing of the data is required after export (letters, reports, etc), select .pdf

  • For backup purposes or to share with a user on another CryptPad instance, select .bin

Presentation application import/export¶

The Презентация supported format are Microsoft PowerPoint .pptx and OpenDocument .odp. These can both be exported from popular software such as Google Slides and LibreOffice Impress:

  1. Export your document as .pptx or .odp in your existing software

  2. Save the file to your computer

  3. Open a new Презентация document (or an existing one to replace the contents)

  4. File > Import and select your file

Для экспорта используйте Файл > Экспорт и выберите формат:

  • To use in Microsoft PowerPoint or Google Slides, select .pptx

  • To use in LibreOffice Impress, select .odp

  • For cases where no further editing of the data is required after export (presentations, reports, etc), select .pdf

  • For backup purposes or to share with a user on another CryptPad instance, select .bin

Spreadsheet application import/export¶

The Таблица supported format are Microsoft Excel .xlsx and OpenDocument .ods. These can both be exported from popular software such as Google Sheets and LibreOffice Calc:

  1. Export your document as .xlsx or .ods in your existing software

  2. Save the file to your computer

  3. Open a new Таблица document (or an existing one to replace the contents)

  4. File > Import and select your file

Для экспорта используйте Файл > Экспорт и выберите формат:

  • To use in Microsoft Excel or Google sheets, select .xlsx

  • To use in LibreOffice Calc, select .ods

  • For cases where no further editing of the data is required after export (presentations, reports, etc), select .pdf

Rich text application import/export¶

The Документ support the HTML format for import/export:

  1. Export your document as HTML in your existing software

  2. Save the file to your computer

  3. Open a new Документ document on CryptPad (or an existing one to replace the contents)

  4. File > Import and select your file

Для экспорта используйте Файл > Экспорт и выберите .doc в качестве формата.

Я не могу сделать X с документом, который я создал¶

Некоторые функции на CryptPad ограничены владельцами документов. Это включает в себя:

  • Enabling an access list

  • Enabling a password

  • Adding or removing other owners

  • Destroying the document

Право собственности на документ устанавливается при создании и не может быть изменено впоследствии, если документ создается без владельца.

Команда разработчиков рассматривает возможность устранения этого различия в будущем. В то же время безопасным вариантом по умолчанию является «владение» всеми созданными вами документами, чтобы сохранить полный контроль над ними, включая возможность их уничтожения.

Могу ли я использовать CryptPad на мобильном телефоне?¶

CryptPad разработан для максимально эффективной работы на маленьких экранах. В зависимости от производительности вашего устройства должно быть возможно использовать CryptPad на мобильном устройстве. В 2020 году была проведена работа по ускорению реакции CryptPad. Если Вы заметили области, которые нуждаются в улучшении в этом отношении, свяжитесь с Техподдержка или отправьте issue на Github.

Вы планируете сделать приложение?¶

Мы не планируем создание специального мобильного приложения по следующим причинам:

  • It would dramatically increase the amount of code that has to be developed and maintained, effectively creating other "versions" of CryptPad for iOS and Android

  • CryptPad is open source and can be hosted by anyone who wants to offer the service. Therefore, users of a mobile application would have to specify which CryptPad instance they want to connect to, which would be confusing. To complicate things further, each instance may be running a different version of the software, depending on whether or not the latest updates were applied by the administrators

Чтобы решить эти проблемы, команда разработчиков работает над превращением CryptPad в "Прогрессивное веб-приложение"("Progressive Web App"). Это означает, что его можно использовать на мобильных устройствах через веб-браузер, и оно будет вести себя как приложение, оставаясь тем же программным обеспечением, которое работает в настольных браузерах. Преимущество этого заключается в том, что каждый экземпляр CryptPad превращается в поставщика веб-приложений, а не возлагает бремя выбора правильного экземпляра на пользователя.

Может ли CryptPad синхронизировать документы с моей локальной файловой системой?¶

Способ шифрования, используемый в настоящее время в CryptPad, не позволяет синхронизировать с локальной файловой системой. Однако эта функция регулярно запрашивается, поэтому команда разработчиков надеется сделать ее доступной в будущем.

Какова связь между CryptPad и OnlyOffice?¶

CryptPad-приложения Документ, Презентация и Таблица являются интеграцией OnlyOffice Docs . Однако, это касается только кода на стороне клиента: CryptPad не использует Cервер Документов OnlyOffice. Зашифрованная совместная работа CryptPad, используемая для документов, презентаций, электронных таблиц и других приложений, полностью отличается от системы шифрования, используемой в некоторых частях OnlyOffice. Некоторые из инструментов преобразования форматов файлов CryptPad основаны на коде OnlyOffice, но была проделана значительная работа, чтобы он работал в браузере, а не на сервере, что позволяет избежать необходимости раскрывать содержимое пользовательских документов при конвертации.

Насколько безопасен CryptPad?¶

CryptPad стремится защитить личность своих пользователей и их контент от нас и от внешних угроз. Однако эта безопасность не является абсолютной и требует, чтобы пользователи соблюдали хорошие практики и чтобы экземпляр, к которому Вы подключаетесь, был надёжным. Пояснения и рекомендации доступны в нашем блоге.

Подведем итоги ключевых моментов:

  • Предполагается, что администраторы экземпляров честны, но любопытны. Это означает, что, хотя они и играют по правилам, они стараются получить как можно больше информации из того, что они могут воспринимать. Мы ведем список публичных экземпляров CryptPad, для которых мы проверили, что они актуальны и хорошо настроены;

  • Каналам связи нельзя доверять, и они могут быть активно вредоносными: внешняя угроза способна подделать, воспроизвести или удалить сообщения. Это не должно повлиять на безопасность CryptPad;

  • Пользователи, с которым Вы делитесь своим документом, должны быть честны, поскольку, как только Вы отправляете свой контент, Вы полагаетесь на них, что они не передадут его третьим лицам. Однако Ваша личность остаётся для них некоторой загадкой: у них есть доступ к Вашему открытому ключу и Вашему отображаемому имени пользователя;

    • Следовательно, ссылку поделиться следует рассматривать как "настолько же конфиденциальную, каковой является кодовая фраза*. Чтобы добавить ещё один уровень безопасности, мы рекомендуем добавить пароль к Вашим файлам. Если у Вас есть учётная запись, они хранятся в Вашем CryptDrive (который считается безопасным), и доступ к ним остаётся беспрепятственным. Однако внешним пользователям, если они каким-то образом найдут ссылку, по-прежнему требуется пароль файла для доступа к нему.

  • Полная история изменений и сведения о различных соавторах видны после публикации документа. Если Вы хотите сохранить это в секрете, лучший способ (на данный момент) — создать копию документа, а затем поделиться ею, имеющей чистую историю изменений.

Часто задаваемые вопросы по cryptpad.fr¶

Могу ли я оплатить подписку без карты?¶

Мы принимаем платежи банковским переводом, Paypal или другим способом в зависимости от запроса. Однако из-за дополнительной работы, связанной с выставлением счетов-фактур и обработкой платежей вручную, этот вариант ограничен годовыми планами для нашего предложения Duo и более дорогих и влечет за собой надбавку в размере 30%.

Принимаете ли вы криптовалюты для подписки и пожертвований?¶

Мы принимаем платежи в биткоинах. На них распространяется действие доплаты за ручную обработку, как объяснено в предыдущем вопросе. Кроме того, как компания, зарегистрированная во Франции, мы по закону обязаны запрашивать имя и адрес для выставления счета, а также адрес электронной почты для сообщения о вашей подписке.

Можете ли вы предоставить Соглашение об обработке данных (DPA)?¶

DPA является частью наших Организационных Планов, обратите внимание, что пример DPA доступен для предварительного просмотра на этой странице.
